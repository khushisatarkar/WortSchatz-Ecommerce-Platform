# WortSchatz: German-Themed E-Commerce Website

WortSchatz is a fully responsive e-commerce website built using HTML-CSS, and JavaScript for a small business selling German-themed merchandise. 
The platform showcases products such as t-shirts, badges, stickers, tote bags, and bookmarks with a focus on clean UI design and smooth user experience.

## 🧩 Features

📱 Fully Responsive Design (Mobile, Tablet, Desktop)

🛒 Product Listing with Grid Layout

🧾 Product Detail Page (Dynamic product loading via ID)

🛍️ Add to Cart Option

🔎 Clean Navigation Bar with Icons

💿 Minimal & Modern UI Design


## 📄 Website Structure

The website consists of multiple interconnected pages to simulate a real e-commerce experience:

- **Home Page** – Featured products and brand introduction
- **Shop Page** – Complete product listing
- **About Page** – Information about the brand and concept
- **Contact Page** – Contact form for customer inquiries
- **Cart Page** – Shopping cart interface for selected items
- **Single Product Page** – Detailed view of individual products

## 🛠️ Tech Stack

HTML5 – Structure of the website

CSS3 – Styling and responsive design

JavaScript – Dynamic functionality and interactions
